Подробнее о принципах работы:

1. Получение запроса от заказчика через колл-центр или электронную почту.

Для того чтобы дать быстрый ответ нам потребуется:

  • Для оценки производства печатных плат — файл проекта, либо gerber файлы, описание технических параметров печатной платы в заполненном бланке заказа.
  • Для оценки монтажа электронного модуля — сборочный чертеж, спецификация на изделие (BOM list) предпочтительные форматы *doc, *xls, *pdf , файл координат установки компонентов.
  • Для оценки сборки в корпус, тестирования, нанесения влагозащитных покрытий — сборочный чертеж на изделие, инструкция по программированию и тестированию.

2. Предварительный анализ и проверка, полученных файлов, уточнение и согласование с заказчиком деталей необходимых для расчета. Согласование допустимых замен по спецификации.

3. Обработка информации и дальнейшее предоставление стоимости и сроков в формате Коммерческого предложения.

4. Согласование условий и сроков поставки. Выставление счета, подписание договора.

5. Закупка комплектующих, подготовка к производству, настройка оборудования для тестирования.

6. Производство изделия.

7. Отгрузка готовой продукции заказчику.
